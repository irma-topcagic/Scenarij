# Scenarij – Collaborative Web Application

## 📌 Description
Scenarij is a collaborative web application for scenario/script writing,
developed as part of university coursework.

## ✨ Features
- Scenario editing
- Line locking for collaborative work
- Versioning using deltas and checkpoints
- Restore previous versions

## 🛠 Technologies
- Node.js
- Express
- JavaScript
- HTML, CSS
- MySQL / Sequelize
